# ジャズピアノ習得ロードマップ（詳細版）

**目標:** ピアノトリオ（ピアノ・ベース・ドラム）でライブ演奏できるレベル
**期間:** 3〜5年
**練習時間:** 1日30分（週3.5時間）
**レッスン予算:** 月1万円以下（対面）
**前提:** ウッドベースでのセッション・ライブ経験あり / コード理論は理解済み / 鍵盤演奏はほぼ未経験

---

## 達成基準の考え方

このロードマップでは、各練習項目に以下の3段階で達成基準を設ける。

| レベル | 定義 | 意味 |
|--------|------|------|
| ✅ 最低ライン | つっかえながらでも正しい音を出せる | 「知っている」段階。次の練習に進んでよいが、並行して定着練習を続ける |
| ⭐ 実用レベル | 指定テンポで止まらずに弾ける | 「できる」段階。セッションで使える最低ライン |
| 🔥 自動化レベル | 考えなくても手が動く。会話しながらでも弾ける | 「身についた」段階。次フェーズの土台として十分 |

**重要：** 各フェーズの「次に進む条件」は⭐実用レベルの達成。🔥は理想だが、全項目で達成を待つ必要はない。

## 現実的な期待値

1日30分 × 365日 = 年間約180時間。3年で約540時間、5年で約900時間。

10,000時間には遠く及ばないが、ベースでのジャズ経験という「貯金」が大きい。理論理解、フォーム感覚、セッション経験、ジャズの耳 — これらは時間に換算できない資産。

**3年後の現実的な到達点：**
知り合いとのカジュアルなライブで、準備した曲をしっかり演奏できるレベル。

**5年後の現実的な到達点：**
セッションで初見の曲もある程度対応でき、自信を持ってトリオライブができるレベル。
