# ボイシングパターンガイド

`PRACTICE_PLAN.md` に分散しているボイシング練習を、フェーズ横断で再整理したページ。  
「今どのボイシングを練習すべきか」を迷わないための実践ガイドとして使う。

---

## このガイドの使い方

- まず現在のフェーズを確認する（Phase 1〜4）
- そのフェーズの「最優先ボイシング」だけを集中して練習する
- 1日30分メニュー内の時間配分は `PRACTICE_PLAN` に従う
- 進行・テンションの理論は [コード進行パターン](/guides/chord-progressions) と併用する

## フェーズ別ロードマップ（抽出版）

| フェーズ | 最優先ボイシング | ゴール |
|---------|------------------|--------|
| Phase 1 | シェルボイシング（3rd/7th）+ ルート追加 | ii-V-I を全12キーで止まらず弾ける |
| Phase 2 | A/Bボイシング（ルートレス）+ Drop 2 + コンピングリズム | 曲上でフォームを瞬時に選択し、伴奏できる |
| Phase 3 | A/Bにテンション追加 + Barry Harris式ブロックコード + インタラクティブ・コンピング | ソリストに反応した立体的な伴奏 |

## 1. シェルボイシング（Phase 1）

### 定義

3度と7度だけでコードの機能を表す最小構成。ジャズピアノの土台。

### Cキー ii-V-I

```text
Dm7:   F + C（3rd + 7th）  または  C + F（7th + 3rd）
G7:    B + F（3rd + 7th）  または  F + B（7th + 3rd）
CMaj7: E + B（3rd + 7th）  または  B + E（7th + 3rd）
```

### ヴォイスリーディングの原則

- `3-7` と `7-3` を交互に使い、手の移動を最小にする
- 例: `Dm7(F+C) -> G7(F+B)`（F共通、Cが半音でBへ）
- 手が大きく飛ぶならフォーム選択が不適切

### 練習手順

1. Cキーのii-V-Iを2パターンで反復
2. F, B♭, E♭を追加
3. 全12キーへ拡張
4. 五度圏で連続運転

### 達成基準

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | 全12キーii-V-Iをメモを見ながら弾ける |
| ⭐ 実用レベル | ♩=60（1コード2拍）で五度圏1周 |
| 🔥 自動化レベル | ♩=100（1コード2拍）でノンストップ |

---

## 2. ルート追加（Phase 1）

### 目的

シェルに低音を足し、実戦で使える厚みを作る。

### パターン

- 左手: ルート単音 + 右手: 3rd & 7th
- または左手で `ルート + 3rd + 7th` の3音

### 達成目安

シェルボイシングの ⭐ 実用レベルと同テンポで安定して弾ける。

---

## 3. A/Bボイシング（ルートレス）（Phase 2）

### 定義

ベースがルートを担当する前提で、右手中心に `3-5-7-9` / `7-9-3-5` で構成する4和音。

### Cキー ii-V-I

```text
【Aフォーム（3-5-7-9）】
Dm7:   F-A-C-E
G7:    B-D-F-A
CMaj7: E-G-B-D

【Bフォーム（7-9-3-5）】
Dm7:   C-E-F-A
G7:    F-A-B-D
CMaj7: B-D-E-G
```

### 練習手順

1. Aフォームのみ全12キー
2. Bフォームのみ全12キー
3. A→B→A→B を五度圏で交互
4. 実曲の進行でフォームを選択

### 達成基準

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | A/B両フォームで全12キーii-V-I |
| ⭐ 実用レベル | ♩=80（1コード2拍）でA/B交互に五度圏1周 |
| 🔥 自動化レベル | 曲上で瞬時にフォーム選択できる |

---

## 4. Drop 2（Phase 2後半〜3）

### 定義

クローズド4和音（4-way close）の「上から2番目の音」を1オクターブ下げるボイシング。

### 例（Cmaj7）

```text
クローズド: C-E-G-B
Drop 2:      G-C-E-B
```

ii-V-I（Cキー）の基本形：

```text
Dm7: A-D-F-C
G7:  D-G-B-F
CMaj7: G-C-E-B
```

### 使いどころ

- 右手コンピングで、クローズドより開いた響きが欲しい時
- メロディをトップノートに置いたまま、内声を滑らかに動かしたい時
- ミディアム〜アップで、4音の輪郭を保って抜けを作りたい時

### 練習手順

1. まずクローズド4和音を作る
2. 上から2番目の音を1オクターブ下げる
3. ii-V-IでA/Bルートレスと弾き比べる
4. 実曲のトップノート固定でDrop 2化する

### 達成基準

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | C, F, B♭, E♭ のii-V-IをDrop 2で弾ける |
| ⭐ 実用レベル | 全12キーでDrop 2を即生成し、進行上で使える |
| 🔥 自動化レベル | A/BルートレスとDrop 2を曲中で瞬時に使い分けられる |

---

## 5. ブロックコード（Barry Harrisメソッド）（Phase 3後半〜4）

### 定義

メロディを最高音に置いた4声体を、Barry Harrisの「6th diminished」の考え方で連続的に作る方法。  
メジャーでは `6th chord` と `dim7` を交互に使うのが基本。

### Cメジャーの基本材料

```text
6th diminished scale: C D E F G A♭ A B
使用コード群: C6系 と Bdim7系（= Ddim7 = Fdim7 = A♭dim7）
```

### 使いどころ

- テーマを厚く harmonize したい時（バラード、ミディアム）
- 単音メロディを「ピアニスティック」に聴かせたい時
- エンディングやフィルでラインに統一感を出したい時

### 練習手順

1. Cメジャーのメロディ8音（ドレミ...）を用意する
2. 各メロディ音をトップにして、C6系 / Bdim7系を交互に当てる
3. 上行・下行を同じ指使いで反復
4. スタンダードの4小節だけに適用して実戦化

### 達成基準

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | Cキーで8音ラインを崩さずブロック化できる |
| ⭐ 実用レベル | 2曲以上でテーマ断片に自然に適用できる |
| 🔥 自動化レベル | 曲想に合わせてブロック化する/しないを即判断できる |

## 6. コンピングリズム（Phase 2）

### 優先パターン

1. チャールストン（1拍目 + 2拍目ウラ）
2. 2拍4拍ウラ打ち
3. シンコペーション（タイ）
4. ランダム組み合わせ

### 練習手順

1. 1コード固定でリズムのみ
2. ii-V-Iでコードチェンジを伴うリズム練習
3. iReal Proで曲通し
4. 好きなピアニストのコンピングを模倣

### 達成基準

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | チャールストンで曲を通せる（♩=100） |
| ⭐ 実用レベル | 3パターン以上を混ぜて♩=140 |
| 🔥 自動化レベル | 伴奏なしでもスイング感を維持 |

---

## 7. テンション追加と応答型コンピング（Phase 3）

### テンション追加

- A/Bボイシングに `9th, 13th, #11th` を追加
- 例: `G7 = F-A-B-E`（`7-9-3-13`）

### インタラクティブ・コンピング

- ソリストのフレーズへ応答する
- ソリスト休符でフィルを入れる
- 盛り上がりに応じて打鍵密度を変える

### 実戦ポイント

- 常に「入れる量」を制御する（詰め込みすぎない）
- セッションでの実践を最優先にする

---

## 8. セッション実践ボイシング（Phase 4）

### イントロで使う形

| パターン | 概要 |
|---------|------|
| ルバート→インテンポ | テンポフリー和音からテンポイン |
| ii-V-I 4小節 | 曲キーのii-V-Iで導入 |
| ペダルポイント | 左手ルート持続 + 右手ハーモニー |

### エンディングで使う形

| パターン | 概要 |
|---------|------|
| タグ・エンディング | 終止2〜4小節を反復 |
| rit. + フェルマータ | 減速して最後を保持 |
| ii-V-I クリシェ | 半音上行/下行で締める |

### 左手ベース併用

- 左手で2ビート（ルート→5度）から開始
- 慣れたらウォーキングへ
- 目標は「左手ベース + 右手コンピング」で1コーラス維持

## ボイシング選択の目安

| 状況 | 優先ボイシング |
|------|----------------|
| まず進行を把握したい | シェル（3rd/7th） |
| セッションで標準的に伴奏したい | A/Bルートレス |
| メロディを厚く harmonize したい | Barry Harris式ブロックコード |
| 開いた4和音で抜けを作りたい | Drop 2 |
| バラードで広がりを出したい | ルート追加 + テンション |
| アップテンポで移動を減らしたい | シェル or ガイドトーン中心 |
| ベースソロ中に支えたい | 左手ベース + 右手コンピング |
