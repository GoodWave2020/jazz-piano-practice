# ロバート・グラスパー奏法ガイド

Robert Glasper 的な「モダンで歌えるグルーヴ」を、再現性のある練習メニューに分解したガイド。  
対象は `Phase 3 後半〜Phase 4`。ビバップ語彙に加えて、現代的なコンピングとライン運用を強化する。

---

## 対象フェーズ

- 推奨開始: `Phase 3` 後半以降
- 前提:
  - [ボイシングパターン](/guides/voicing-patterns) のシェル/A-Bフォームを使える
  - [ビバップ特化練習](/guides/bebop-practice) の「拍頭着地」を理解している
  - 2拍4拍クリックで1コーラスのコンピングを維持できる

---

## 奏法の軸（4つ）

1. **ゴスペル由来の厚いボイシング**  
   3rd/7th を芯に、9th・11th・13th を加えて色を作る。
2. **後ノリのコンピング配置**  
   「拍の後ろ」に置く意識で、急がないタイムを作る。
3. **短いモチーフの反復とずらし**  
   長い速いラインより、短い素材を変形して展開する。
4. **スイングとストレートの往復**  
   曲想に応じて8分の質感を切り替える。

---

## 1. コアドリル

### 1-1. ゴスペル系スプレッド・ボイシング

右手は4音の塊、左手は最小限のルートで支える。

<AbcNotation abc='
X:1
T:Gospel Spread (ii-V-I in C)
M:4/4
L:1/8
K:C
V:1 clef=treble name="R.H."
"Dm11"[face]2 z2 [cefa]2 z2 | "G13"[bdfa]2 z2 [abdf]2 z2 | "Cmaj9"[egbd]8 |
V:2 clef=bass name="L.H."
D,4 D,4 | G,,4 G,,4 | C,8 |
' />

ドリル:
1. まず `Dm11-G13-Cmaj9` だけで2分ループ
2. 右手の和音形を1種類固定し、タイムの安定を優先
3. 12キー移調は「白鍵キー4つ」から始める

### 1-2. 後ノリ・コンピング配置

オフビート中心で、密度を上げすぎずにノリを作る。

<AbcNotation abc='
X:1
T:Offbeat Comping Placement
M:4/4
L:1/8
K:C
V:1 clef=treble
z [egbd] z [egbd] z [dfac] z [dfac] |
z [dfac] z [cegb] z [cegb] z [dfac] |
V:2 clef=bass
C,8 | F,8 |
' />

ドリル:
1. メトロノームは `2拍4拍`
2. まず音量 `mp` 固定で、打鍵の深さをそろえる
3. うまくハマった小節だけを4回反復してから通す

### 1-3. モチーフ反復 + リズムずらし

同じ素材を「位置だけ」変えて、モダンな流れを作る。

<AbcNotation abc='
X:1
T:Motif Displacement
M:4/4
L:1/8
K:C
"Dm7"a c d c a c d c | z a c d c a c d |
' />

ドリル:
1. 4音モチーフを1個だけ作る
2. 2小節目は先頭に8分休符を入れてずらす
3. コードだけ変えて同型運用（例: Dm7→G7→Cmaj7）

### 1-4. スイング/ストレート切替ドリル

1コーラス内で無理に混ぜず、`Aセクション=スイング / Bセクション=ストレート` のように大きく切り替える。

ドリル:
1. まず片方のフィールだけで通す
2. 次にセクション単位で切替
3. 録音して「急に速く聞こえる箇所」を特定する

### 1-5. ペダルポイントでメジャー感→マイナー感

同じ低音を維持したまま、右手だけで空気を切り替える。

ドリル:
1. 左手は `F` ペダル固定
2. 前半2小節は `Ebmaj7/F` 系、後半2小節は `Fm7` 系へ切替
3. 切替の瞬間にテンポが揺れないか録音で確認

### 1-6. オスティナート + 下降ベース

右手反復は維持し、左手だけ下降させる。

ドリル:
1. 右手モチーフを1つ固定
2. 左手を `D -> C -> B -> Bb` と4小節で下降
3. 右手が崩れたら、前半2拍だけ完全固定で再構築

### 1-7. 移動トライアド

機能和声より「同じ質感の移動」を優先する。

ドリル:
1. `Gm -> Dsus系 -> C9系 -> Cm9系` を4小節で並べる
2. 右手トップノートを歌いながら弾く
3. 同型のままキーを移す

### 1-8. 2度クラスター固定

下2声の2度を芯にして、濁りの質を作る。

ドリル:
1. `F-G + C-E`（Dm11/G13sus）を `mp` で反復
2. 解決先だけ `E-F# + B-D` に切替（Cmaj9#11）
3. 低域で濁る場合は右手を1オクターブ上げる

### 1-9. 8分ライン + 左手m9下降

右手ラインと左手半音下降を同時維持する。

ドリル:
1. 左手 `Em9系 -> Ebm9系 -> Dm9系 -> Dbm9系`（2拍ずつ）
2. 右手は拍頭コードトーン、拍裏クロマチック
3. 小節末に短い休符を入れて呼吸を作る

## 2. グラスパー系コード進行テンプレ

ここでは「Glasper 作品に多く見られる傾向」を、練習用テンプレートとして抽出する。  
厳密コピーではなく、現場で使える形に単純化している。

### 2-1. マイナー・ペダル + ベース下降

右手の形を固定し、左手ベースだけを半音/全音で動かす。

<AbcNotation abc='
X:1
T:Minor Pedal with Descending Bass
M:4/4
L:1/4
K:C
V:1 clef=treble name="R.H."
"Dm11"[face]4 | "Dm11/C"[face]4 | "Dm11/B"[face]4 | "Dm11/Bb"[face]4 |
V:2 clef=bass name="L.H."
D,4 | C,4 | B,,4 | _B,,4 |
' />

狙い:
- コード名が変わっても、上声のキャラクターを維持する
- ベースの線で緊張を作る

### 2-2. `sus` チェーン → 解決

機能和声の「V感」を残しつつ、濁りをコントロールする。

| 進行 | 使い方 |
|------|--------|
| `Fm9 - Bb13sus - Ebmaj9` | ゴスペル寄りの着地 |
| `Dm9 - G13sus - Cmaj9` | ii-V-I をモダン化 |
| `Dm9 - Db13sus - Cmaj9` | 半音アプローチで色を強める |

狙い:
- 3rd を急いで出さず `sus` の浮遊感を先に作る
- 解決先でだけ明確に 3rd を聞かせる

### 2-3. 2コード・ヴァンプ（4〜8小節反復）

短い反復で、リズムと音色の変化を鍛える。

| ヴァンプ | 練習テーマ |
|---------|-----------|
| `Cm9 - Abmaj7(#11)` | モーダルな横移動 |
| `Em11 - A13sus` | 16分系グルーヴへの適応 |
| `F#m11 - Emaj9` | 低音ペダル上のボイシング差分 |

ドリル:
1. まず `8小節` 同じリズムで反復
2. 次の `8小節` はリズムだけ変えて和音形は固定
3. 最後の `8小節` で和音形を1つだけ差し替える

### 2-4. 記事由来の進行テンプレ（4小節）

| タイプ | 進行 | 用途 |
|------|------|------|
| ペダル切替 | `Fペダル + (Ebmaj7/F -> Fm7系)` | 同一低音で色を切り替える |
| 下降ベース | `D -> C -> B -> Bb`（右手反復固定） | 反復内に緊張を作る |
| m9下降 | `Em9系 -> Ebm9系 -> Dm9系 -> Dbm9系` | ラインと半音推進を同時強化 |

## 3. グラスパー系ボイシングテンプレ

### 3-1. 左手ミニマム + 右手クラスター

左手はルート（またはルート+5度）だけ、右手で色を作る。

| コード | 左手 | 右手（例） | 役割 |
|------|------|------------|------|
| Dm11 | `D` | `F-C-E-G` | 3rd/7th を芯に11thまで含む |
| G13sus | `G` | `F-A-C-E` | 3rdを抜いて浮遊感を維持 |
| Cmaj9(#11) | `C` | `E-B-D-F#` | 明るいLydianカラー |

### 3-2. `sus` から 3rd へ遅延解決

同じベース上で「sus → ドミナント」を切り替えて推進力を作る。

<AbcNotation abc='
X:1
T:Sus to Dominant Resolve (G7)
M:4/4
L:1/8
K:C
V:1 clef=treble name="R.H."
"G13sus"[face]2 [face]2 "G7(b9)"[bdf_a]2 [bdf_a]2 | "Cmaj9"[egbd]8 |
V:2 clef=bass name="L.H."
G,,8 | C,8 |
' />

ポイント:
- 前半は 3rd を避ける
- 解決直前だけ 3rd（B）を含む形へ切替える

### 3-3. 平行移動（Planing）で色を保つ

機能和声より「音色の連続性」を優先する場面で有効。

ドリル:
1. 右手4音形を1つ作る（例: `F-C-E-G`）
2. そのまま半音上/下に平行移動
3. 左手はペダルまたは単純な下降ベースを維持

注意:
- 1コーラス全体で多用しない（`4小節` 程度に限定）
- メロディと衝突する場合はトップノート優先で修正

### 3-4. 記事由来クラスター運用ルール

| ルール | 内容 |
|------|------|
| ダイナミクス | まず `mp` 固定。濁りを制御してから音量を上げる |
| 音域 | 低域で濁る場合は右手を1オクターブ上へ |
| 運用長 | 連続使用は4小節以内。以降は開いた形へ戻す |

## 6. ありがちな失敗

- **和音を増やしすぎて重くなる**  
  対策: 1小節の発音回数を4回以下に制限する
- **後ノリのつもりで遅れる**  
  対策: 左手だけ先にクリックへ合わせ、右手を後ろへ置く
- **モチーフが毎回別物になる**  
  対策: 音高は固定し、まずリズムだけ変える
- **ペダル切替でテンポが揺れる**  
  対策: 切替直前に8分休符を1つ入れて再侵入を安定させる
- **2度クラスターが痛く聞こえる**  
  対策: `mp` 固定 + 右手を高めの音域へ移動
- **m9下降で前のめりになる**  
  対策: 左手のみを2拍4拍クリックで先に固定する
