# ピアニスト定番奏法ガイド

「ピアニストっぽい表現」を、再現性のある練習メニューとして整理したガイド。  
対象は Phase 3 後半〜Phase 4。バラードやイントロ/エンディングで特に効果が高い。

---

## このガイドの使い方

- まず [PRACTICE_PLAN の30分配分](/PRACTICE_PLAN#補助メニューピアニスト定番奏法phase-3後半) に従って、週3回・10分で実施
- 練習は常に「遅いテンポから」。速さより音色・着地音・タイムを優先
- 1つの曲に対して、同時に複数の奏法を入れすぎない（1コーラス1〜2個まで）

---

## 共通ルール

1. メトロノームを使う（バラードでもクリックを消さない）
2. 全奏法で「弱めの音量」から開始する（脱力優先）
3. うまくいかない日はテンポを10下げる
4. 週1回だけ録音して、音量バランスとタイムを確認する

---

## 1. ハンドオーバーハンド（クロスハンド）・アルペジオ

### 目的

- イントロやエンディングで、音域の広がりを作る
- バラードの「立ち上がり」を印象的にする

### 2分ドリル

1. Cmaj7 を1パターン固定で上行2回、下行2回
2. テンポは ♩=50〜60、8分音符
3. 両手交差の直前だけ視線を移動し、着地音を確実に当てる

### 譜例（クロスハンド・アルペジオ）

<AbcNotation abc='
X:1
T:Cross-hand Arpeggio (LH4 -> RH4)
M:4/4
L:1/16
Q:1/4=56
K:C
V:1 clef=treble
"L.H."C,,, G,,, C,, E,, "R.H."G,, C, E, G, "L.H."C E G c "R.H."e g c&apos; e&apos; |
"R.H."e&apos; c&apos; g e "L.H."c G E C "R.H."G E C, G, "L.H."E, C, G,, C,, |
' />

※画像の例に合わせた「4音ずつ手を交代して上行→下行する」簡略譜。必要に応じて最高音をさらに上げる。

### 失敗しやすい点

- 交差の瞬間にアクセントが強くなる  
対処: 交差直前の音を少し弱くして、次音を同じ音量でつなぐ
- ルート着地が曖昧になる  
対処: 最後の1音だけ左手で単音確認して終了する

### 達成基準

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | C, F, B♭, E♭ で1パターンを崩れず弾ける |
| ⭐ 実用レベル | バラードのイントロに1回だけ自然に挿入できる |
| 🔥 自動化レベル | 曲想に合わせて上行/下行を即選択できる |

---

## 2. オクターブ・ユニゾン

### 目的

- メロディの輪郭を太くする
- クライマックスのエネルギーを作る

### 2分ドリル

1. テーマ冒頭4小節をオクターブで弾く
2. 右手主導でレガートを作り、左手は補助に徹する
3. 2拍4拍に軽いアクセントを置く

### 譜例（オクターブ・ユニゾン）

<AbcNotation abc='
X:1
T:Octave Unison Line
M:4/4
L:1/8
K:C
V:1 clef=treble name="R.H."
"Cmaj7"e2 g2 a2 g2 | "Dm7"f2 e2 d2 c2 |
V:2 clef=bass name="L.H."
C,2 E,2 F,2 E,2 | D,2 C,2 B,,2 A,,2 |
' />

### 失敗しやすい点

- 手首が固まってタイムが突っ込む  
対処: 1小節ごとに脱力チェック
- 音量が大きすぎる  
対処: mp〜mf の範囲で固定する

### 達成基準

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | 4小節を均一な音量で弾ける |
| ⭐ 実用レベル | 8小節をテンポキープして弾ける |
| 🔥 自動化レベル | 曲の山だけに絞って自然に使える |

---

## 3. ロールド・コード（分散和音）

### 目的

- バラードで和音の立ち上がりを柔らかくする
- ルバート導入の空気感を作る

### 2分ドリル

1. Imaj7 と iim7 を下から上にロール
2. まず3音、次に4音へ拡張
3. 拍頭は必ずベース音が先に聞こえるようにする

### 譜例（ロールド・コード）

<AbcNotation abc='
X:1
T:Rolled Chords (ii-V-I)
M:4/4
L:1/8
K:C
V:1 clef=treble name="R.H."
"Dm9"f a c e "G13"f a b e | "Cmaj9"e g b d e2 d2 |
V:2 clef=bass name="L.H."
D,4 G,,4 | C,4 C,4 |
' />

### 失敗しやすい点

- ただの「ばらけた和音」になる  
対処: ベース音から上声までの時間差を一定にする
- 毎回同じ速さで機械的になる  
対処: 小節頭だけ遅め、弱拍は速めの2パターンを作る

### 達成基準

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | ii-V-I で拍頭を濁らせずにロールできる |
| ⭐ 実用レベル | イントロ/エンディングで安定して使える |
| 🔥 自動化レベル | コードタイプごとにロール速度を即調整できる |

---

## 4. ブロックコード・フィル

### 目的

- メロディ終端の余白を自然に埋める
- ソロや伴奏の返しに短いアクセントを作る

### 2分ドリル

1. 2〜4音の短いメロディ断片を選ぶ
2. 断片だけ和音化して、1コーラス1回挿入
3. 休符の位置にだけ入れる

### 譜例（ブロックコード・フィル）

<AbcNotation abc='
X:1
T:Block Chord Fill Example
M:4/4
L:1/8
K:C
V:1 clef=treble name="R.H."
"Cmaj7"e2 g2 z4 | "A7"[ceg]2 [dfa]2 "Dm7"d2 z2 | "G7"[bdf]2 [ace]2 "Cmaj7"c4 |
V:2 clef=bass name="L.H."
C,8 | A,,8 | G,,4 C,4 |
' />

### 失敗しやすい点

- 入れすぎて歌が消える  
対処: 1コーラス2回まで
- タイミングが後ろに落ちる  
対処: 先にリズムだけ手拍子で確認してから弾く

### 達成基準

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | 1回のフィルを崩さず挿入できる |
| ⭐ 実用レベル | 曲の流れを壊さず2回まで使える |
| 🔥 自動化レベル | ソリストの休符に反応して即挿入できる |

---

## 5. エフェクト（短いグリッサンド / トレモロ）

### 目的

- イントロやエンディングの「ここぞ」で印象を作る
- ダイナミクスの頂点を明確にする

### 2分ドリル

1. 単独で音量を一定にして出す
2. 直後の着地コードまでをワンセットで練習
3. 「曲中1回」ルールで通し練習

### 譜例（グリッサンド / トレモロ）

<AbcNotation abc='
X:1
T:Effect Landing Example
M:4/4
L:1/16
K:C
V:1 clef=treble name="R.H."
"gliss"cdefgabccbagfedc | "trem"c c c c c c c c "Cmaj7"[ceg]8 |
V:2 clef=bass name="L.H."
C,16 | G,,8 C,8 |
' />

### 失敗しやすい点

- 効果音だけが浮く  
対処: 直後の着地コードを先に決める
- 手首や指に負担が出る  
対処: 反復回数を減らし、痛みが出たら即中止する

### 達成基準

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | 1種類を安定して再現できる |
| ⭐ 実用レベル | 曲中1回で自然に着地できる |
| 🔥 自動化レベル | 曲想に応じて使う/使わない判断ができる |

## 曲への適用例

| 曲 | 使う奏法 | 入れる場所の目安 |
|----|----------|------------------|
| Body and Soul | ロールド・コード、クロスハンド・アルペジオ | イントロ冒頭、エンディング直前 |
| Misty | オクターブ・ユニゾン、ブロックコード・フィル | テーマ後半の山、終止直前 |
| In a Sentimental Mood | ロールド・コード、短いトレモロ | ルバート部、最後のロングトーン |
| Autumn Leaves | ブロックコード・フィル | Aセクション終端の休符 |
| Blue Bossa | オクターブ・ユニゾン（控えめ） | テーマ再現部の山のみ |

## 週間ローテーション例（週3回）

| 日 | 10分メニュー |
|----|-------------|
| Day 1 | クロスハンド 2分 / ロールド 2分 / オクターブ 2分 / フィル 2分 / エフェクト 2分 |
| Day 2 | オクターブ 2分 / フィル 2分 / ロールド 2分 / クロスハンド 2分 / エフェクト 2分 |
| Day 3 | 曲に適用 10分（1曲で2奏法まで） |

## 実戦前チェック

- どの奏法も「曲中の使いどころ」が説明できる
- 1曲で使う奏法数を決めてから弾いている
- 録音で「やりすぎ」に聞こえない
- 痛みや違和感がない状態で弾ける
