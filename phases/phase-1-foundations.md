# Phase 1：鍵盤の基礎固め（0〜6ヶ月）

## レッスンの活用法

| 時期 | 頻度 | レッスンで見てもらうこと | 独学で進めること |
|------|------|----------------------|---------------|
| 0〜3ヶ月 | 月2回 | フォーム矯正、脱力チェック、フィンガリング確認 | スケール反復、ボイシング暗記、曲の譜読み |
| 3〜6ヶ月 | 月1回 | 変な癖がついていないかチェック、次の練習課題の方向性確認 | 同上の継続 |

**Phase 1でレッスンが必要な理由：**
鍵盤のフォームと脱力は、文章や動画では正確に伝わらない。力が入ったまま練習を続けると、腱鞘炎のリスクがあるだけでなく、速いパッセージが物理的に弾けなくなる。ここだけは最初に対面で見てもらう価値がある。

**レッスン前の準備（毎回）：**
- 今練習している内容を先生の前で弾けるように準備する
- 「ここが上手くいかない」という具体的な質問を用意する
- 漠然と「教えてください」で行かない

## 1-1. スケール・基礎テクニック（10分）

### メジャースケール

**やること：** 両手ユニゾン（1オクターブ）でメジャースケールを弾く

**進め方：**
1. 最初の2週間：C, F, Bb, Eb の4キーのみ
2. 3〜4週目：Ab, Db, Gb を追加
3. 5〜8週目：G, D, A, E, B を追加して全12キー
4. 全キーが弾けたら2オクターブに拡張

**フィンガリング（指番号）：**
- Cメジャーの右手：1-2-3-1-2-3-4-5（親指くぐりが最重要テクニック）
- キーごとに正しい指番号があるので、先生に最初に確認すること
- **絶対に自己流のフィンガリングで練習しない**（後から矯正が極めて困難）

**練習のやり方：**
- メトロノームを必ず使う
- 1音＝1拍（4分音符）で弾く
- 各音の長さ・音量を揃えることを最優先する（速さは二の次）

**達成基準：**

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | 全12キーで正しい音・正しい指番号で弾ける（テンポ不問、止まってOK） |
| ⭐ 実用レベル | 全12キーを♩=80で4分音符、ノンストップで弾ける（音量のバラつきが少ない） |
| 🔥 自動化レベル | 全12キーを♩=120で8分音符、スムーズに弾ける |

---

### ナチュラルマイナースケール（Phase 1後半〜）

**やること：** メジャースケールと同じ要領でナチュラルマイナースケール全12キー

**進め方：**
- メジャースケールが⭐実用レベルに達してから開始
- メジャーと同じ順番（Am, Dm, Gm, Cm...）で進める

**達成基準：** メジャースケールと同じ基準を適用

---

## 1-2. ボイシング練習（10分）

### シェルボイシング（3rd & 7th）

**シェルボイシングとは：**
コードの3度と7度だけを左手で弾くボイシング。最小限の音でコードの性格（メジャー/マイナー/ドミナント）を表現できる、ジャズピアノの最も基本的なボイシング。

**押さえ方（Cキーのii-V-Iの場合）：**

```
Dm7:  F + C（3rd + 7th）  または  C + F（7th + 3rd）
G7:   B + F（3rd + 7th）  または  F + B（7th + 3rd）
CMaj7: E + B（3rd + 7th） または  B + E（7th + 3rd）
```

**ヴォイスリーディングの原則：**
- 上の2パターン（「3-7」と「7-3」）を交互に使い、手の移動を最小限にする
- Dm7を「F+C」で弾いたら → G7は「F+B」（Fが共通、Cが半音下がってB）
- **手が大きく飛ぶ場合はパターンの選択が間違っている**

**練習の進め方：**
1. 最初の2週間：Cキーのii-V-I（Dm7→G7→CMaj7）を2パターンで
2. 3〜4週目：F, Bb, Eb キーを追加（♭系から始めるのはジャズの慣例）
3. 5〜8週目：全12キーのii-V-I
4. 8週目〜：ii-V-I を五度圏（C→F→Bb→Eb→...）で連続して弾く

**メトロノームの使い方：**
- 1コード＝2拍（♩=60）で切り替え
- 慣れたら1コード＝1拍で切り替え

**達成基準：**

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | 全12キーのii-V-Iを楽譜やメモを見ながら弾ける（テンポ不問） |
| ⭐ 実用レベル | 全12キーのii-V-Iを♩=60（1コード2拍）でノンストップ。五度圏で一周できる |
| 🔥 自動化レベル | 五度圏を♩=100（1コード2拍）でノンストップ。コード名を言われたら即座に押さえられる |

---

### ルート追加（右手 or 左手）

**やること：** シェルボイシングにルート音を追加して、よりフルなサウンドにする

**パターン：**
- 左手：ルート（単音） ＋ 右手：3rd & 7th
- または左手で「ルート＋3rd＋7th」を3音で押さえる

**達成基準：** シェルボイシングの⭐実用レベルと同じテンポで弾ければOK

## 1-3. 曲の練習（10分）

### 最初に取り組む曲（優先順）

1. **Autumn Leaves**（最優先。ベースで弾いたことがあるはず）
2. **Blue Bossa**（ボサノバだがコード進行がシンプル）
3. **Now's the Time**（Fブルース。12小節で短い）
4. **Fly Me to the Moon**
5. **All of Me**

### 練習のステップ（1曲あたり）

**Step 1：左手ボイシングだけで通す**
- シェルボイシングでコード進行を通して弾く
- iReal Proのテンポを♩=80〜100に落として合わせる
- **ゴール：** 曲のフォーム（AABA等）に沿って、止まらずにコードチェンジできる

**Step 2：右手メロディだけで弾く**
- リードシート（メロディ+コード記号の楽譜）を見て右手でメロディを弾く
- 正しいリズムで弾く（音が合っていてもリズムが違えば×）
- **ゴール：** 元のテンポの70%で止まらずにメロディを弾ける

**Step 3：両手合わせ**
- 左手ボイシング＋右手メロディ
- 最初はテンポを大幅に落とす（元テンポの50%でOK）
- **ゴール：** 元テンポの70%で通して弾ける

**Step 4：iReal Proと合わせる**
- マイナスワン（ベース＋ドラム）と一緒に弾く
- **ゴール：** 原曲テンポで止まらずに2コーラス弾ける

### 達成基準（1曲あたり）

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | Step 2まで完了（両手はまだバラバラでもOK） |
| ⭐ 実用レベル | Step 4完了。原曲テンポの70%以上で、止まらずに2コーラス弾ける |
| 🔥 自動化レベル | 原曲テンポで弾ける。リードシートを見なくても1コーラスは弾ける |

### 「弾ける」の定義（全フェーズ共通）

以下を全て満たして初めて「弾ける」とする：
1. **止まらない** — テンポが遅くても、曲の頭から終わりまで止まらずに弾き通せる
2. **拍を見失わない** — ミスをしても次の小節の頭で復帰できる（ベースでの経験が活きる）
3. **フォームを把握している** — 今自分が曲のどこにいるか分かっている
4. **コードが鳴っている** — 単にメロディだけでなく、ハーモニーが聞こえる
