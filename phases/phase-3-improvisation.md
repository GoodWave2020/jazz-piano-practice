# Phase 3：アドリブとコンピングの実践（18〜30ヶ月）

## レッスンの活用法

| 頻度 | レッスンで見てもらうこと | 独学で進めること |
|------|----------------------|---------------|
| 月2回 | アドリブの質の向上、表現力の深化、先生とのデュオ実践 | スケール・アルペジオ反復、リック暗記、トランスクリプション |

**Phase 3 でレッスンの価値が最大化する理由：**
アドリブの「音楽的な質」は独学では判断しにくい。「音は合っているけど面白くない」「フレーズがスケール練習に聞こえる」といった問題は、先生にリアルタイムで聴いてもらい指摘してもらうのが最も効率的。

**レッスンでやるべきこと：**
- 自分のアドリブを先生の前で弾く → フィードバックをもらう
- 先生とデュオで演奏する（先生がベース/コンピング、自分がソロ）
- 先生にアドリブを弾いてもらい、アプローチの違いを体感する
- 「この曲のここでいつも詰まる」という具体的な相談

**レッスンでやらなくていいこと：**
- スケールやアルペジオの練習（独学でやる）
- リックの暗記（独学でやる）
- 曲の譜読み（独学でやる）

## 3-1. テクニック（5分）

### アルペジオ

**やること：** 各コードタイプ（Maj7, m7, 7, m7♭5）のアルペジオを全12キーで弾く

**練習の進め方：**
1. 1オクターブ上行→下行（右手のみ → 左手のみ → 両手）
2. 慣れたら2オクターブに拡張
3. ii-V-I進行でアルペジオを繋げて弾く

**達成基準：**

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | 4種のコードタイプ × 全12キーで正しい音を弾ける |
| ⭐ 実用レベル | ♩=100の8分音符で全キーをノンストップ |
| 🔥 自動化レベル | ii-V-I進行上でアルペジオを切り替えながら♩=140で弾ける |

---

### 発展的スケール

Phase 2のスケールに加えて：

| スケール | 用途 | 練習方法 |
|---------|------|---------|
| オルタードスケール | V7上でテンション感を出す | まず♭9, ♯9, ♯11, ♭13 の響きを耳で覚える → スケールとして弾く |
| コンビネーション・オブ・ディミニッシュ | V7上の別のアプローチ | 半音-全音の交互パターンで全12キー |
| ホールトーンスケール | V7上の浮遊感。Monk的サウンド | 全音間隔なので実質2パターンのみ。習得は速い |
| ビバップドミナントスケール | 8分音符のラインでコードトーンを拍頭に揃える | ミクソリディアン＋M7の8音スケール |
| リディアン♭7 | サブドミナントマイナー系 | Phase 3後半〜 |

### 各スケールの譜例（すべてCスタート）

**譜面の読み方：** ト音記号譜（右手）の記号＝**スケール度数**、ヘ音記号譜（左手）の数字＝**指番号**

**オルタードスケール（Cオルタード）：** D♭メロディックマイナーの第7モード。V7上で**♭9, ♯9, ♯11(♭5), ♭13** のテンションを全て含む最も緊張感の高いスケール。

<AbcNotation abc='
X:1
T:Altered Scale (C)
M:4/4
L:1/4
K:C
V:1 clef=treble name="度数"
"R"C "b9"_D "#9"_E "3"_F | "b5"_G "b13"_A "b7"_B "R"c |
V:2 clef=bass name="指番号"
"4"C, "3"_D, "2"_E, "1"_F, | "4"_G, "3"_A, "2"_B, "1"C |
' />

全音が♭している異様な見た目だが、「D♭メロディックマイナーをCから弾く」と覚えれば12キー対応が楽。♭Fは異名同音でEナチュラル（=メジャー3rd）。左手は**4-3-2-1**の繰り返し。親指がF♭(=E)とC（ともに白鍵）に着地する。

**コンビネーション・オブ・ディミニッシュ（Cコンディミ / H-W Diminished）：** **半音-全音**が交互に並ぶ8音スケール。V7上でオルタードとは違う角度のテンション感を出す。

<AbcNotation abc='
X:1
T:H-W Diminished Scale (C)
M:4/4
L:1/4
K:C
V:1 clef=treble name="度数"
"R"C "b9"_D "#9"_E "3"=E | "#11"^F "5"G "13"A "b7"_B | "R"c2 z2 |
V:2 clef=bass name="指番号"
"4"C, "3"_D, "2"_E, "1"=E, | "4"^F, "3"G, "2"A, "1"_B, | "4"C2 z2 |
' />

8音スケールは対称形のため、実質**3パターンのみ**（C=E♭=G♭=A、D♭=E=G=B♭、D=F=A♭=B）。1つ覚えれば4キー分。左手も**4-3-2-1**の繰り返しパターン。

**ホールトーンスケール（Cホールトーン）：** 全音間隔のみで構成される6音スケール。独特の浮遊感・夢のような響き。Thelonious Monkが多用。

<AbcNotation abc='
X:1
T:Whole Tone Scale (C)
M:4/4
L:1/4
K:C
V:1 clef=treble name="度数"
"R"C "2"D "3"E "#4"^F | "#5"^G "#6"^A "R"c2 |
V:2 clef=bass name="指番号"
"3"C, "2"D, "1"E, "4"^F, | "3"^G, "2"^A, "1"C2 |
' />

完全な対称形のため、**実質2パターンのみ**（C=D=E=F♯=G♯=A♯ と D♭=E♭=F=G=A=B）。半音ずらせば全キー対応。左手は**3-2-1-4-3-2-1**。親指がE（白鍵）とC（白鍵）に着地する。

**ビバップドミナントスケール（Cビバップドミナント）：** ミクソリディアンに**メジャー7th（B♮）を経過音として追加**した8音スケール。8分音符で弾くとコードトーン（R, 3, 5, ♭7）が全て拍頭に来る。

<AbcNotation abc='
X:1
T:Bebop Dominant Scale (C)
M:4/4
L:1/4
K:C
V:1 clef=treble name="度数"
"R"C "2"D "3"E "4"F | "5"G "6"A "b7"_B "7"=B | "R"c2 z2 |
V:2 clef=bass name="指番号"
"5"C, "4"D, "3"E, "2"F, | "1"G, "4"A, "3"_B, "2"=B, | "1"C2 z2 |
' />

♭7（B♭）とナチュラル7th（B♮）が半音で並ぶのが特徴。下行で「C-B♮-B♭-A-G...」と弾くとビバップらしいクロマチックなラインになる。左手は**5-4-3-2-1-4-3-2-1**。G（白鍵）で親指くぐりの後、4-3-2-1で追加音をカバーする。

**リディアン♭7スケール（Cリディアン♭7）：** メロディックマイナーの第4モード。リディアン（♯4）とミクソリディアン（♭7）を合わせたような響き。サブドミナントマイナー系や♯11系コード上で使用。

<AbcNotation abc='

T:Lydian b7 Scale (C)
M:4/4
L:1/4
K:C
V:1 clef=treble name="度数"
"R"C "2"D "3"E "#4"^F | "5"G "6"A "b7"_B "R"c |
V:2 clef=bass name="指番号"
"5"C, "4"D, "3"E, "2"^F, | "1"G, "3"A, "2"_B, "1"C |
' />

リディアンとの違いは**7度だけ**（7 vs ♭7）。ミクソリディアンとの違いは**4度だけ**（4 vs ♯4）。左手は**5-4-3-2-1-3-2-1**（Cメジャーと同じ）。

**達成基準：** ドリアン・ミクソリディアンと同じ基準を適用（まず4キー → 全12キー）

## 3-2. アドリブ練習（10分）

### コードトーン・ソロ

**やること：** コードの構成音（1-3-5-7）だけを使ってソロを取る

**なぜコードトーンから始めるか：**
スケールでのアドリブは「それっぽいけど何を弾いているか分からない」状態になりがち。コードトーンを軸にすることで、コード進行がソロに反映される「ジャズらしい」ソロになる。

**練習の進め方：**
1. 曲のコード進行上で、各コードの1-3-5-7を順番に弾く（機械的でOK）
2. リズムを変える（全音符→2分→4分→シンコペーション）
3. コードトーンの間を経過音（パッシングトーン）で埋める
4. コードトーン＋覚えたリックを組み合わせる

**iReal Proでの練習：**
- テンポを♩=100以下に落として始める
- 1コーラス通して弾く→録音→聴き返す→改善、のサイクル

**達成基準：**

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | Autumn Leavesのコード進行上でコードトーンを追って弾ける（♩=80） |
| ⭐ 実用レベル | レパートリーの曲上で、コードトーン+経過音を使って2コーラスのソロが取れる（♩=120） |
| 🔥 自動化レベル | コードトーン・スケール・リックを自由に混ぜて、音楽的なソロが弾ける |

---

### スケール・ベースのアドリブ

**やること：** コードに対応するスケールを使ってアドリブする

**コードとスケールの対応（基本）：**

| コードタイプ | 使うスケール |
|------------|------------|
| Maj7 | イオニアン or リディアン |
| m7 | ドリアン |
| 7 | ミクソリディアン → 慣れたらオルタード |
| m7♭5 | ロクリアン |

**練習の進め方：**
1. ii-V-I でスケールを切り替えながら8分音符で弾く（機械的に上行→下行）
2. フレーズの始まりをコードトーンにする
3. リズムにバリエーションをつける
4. 「歌うように弾く」 — 弾く前に頭の中でフレーズを歌う練習

**達成基準：**

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | 曲上でスケールを切り替えながら弾ける（♩=80、機械的でもOK） |
| ⭐ 実用レベル | ♩=120でスケールとコードトーンを混ぜた自然なラインが弾ける |
| 🔥 自動化レベル | リズム・ダイナミクスに変化をつけながら、ストーリーのあるソロが取れる |

### コンピングの発展

**テンションの追加：**
- A/Bボイシングに9th, 13th, #11th を加える
- 例：G7 で「F-A-B-E」（7-9-3-13）

**インタラクティブ・コンピング：**
- ソリストのフレーズに「応答する」ようなコンピング
- ソリストが休んだらフィルを入れる、盛り上がりに合わせてリズムを密にする
- **セッションでの実践が最大の練習場**
