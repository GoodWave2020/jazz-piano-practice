# Phase 4：セッションホスト運営力（30〜42ヶ月）

## レッスンの活用法

| 頻度 | レッスンで見てもらうこと | 独学で進めること |
|------|----------------------|---------------|
| 月1〜2回 | セッション進行の設計、キー/テンポ設定、キュー出し、現場トラブル対応 | 模擬ホスト進行、レパートリー整備、テクニック維持 |

**Phase 4 でのレッスンの使い方：**
この段階ではレッスンの役割が「教わる」から「相談する」に変わる。演奏技術だけでなく、曲順設計、参加者のレベル差への対応、時間配分といったホスト実務を具体化する。

**先生にお願いできると理想的なこと：**
- 模擬セッション形式で「曲振り→カウント→ソロ回し→終わり方」まで通しで見てもらう
- 先生のセッション現場で、ホストの声かけ・進行を観察する
- 実際のセッションでホスト代行を短時間でも経験させてもらう

**レッスン頻度を月1回に減らせるタイミング：**
進行テンプレートが固まり、実地で月1回以上回せるようになったら、レッスンはメンテナンス的な月1回で十分。

| 時間 | 内容 |
|------|------|
| 5分 | テクニック維持 |

---

## 4-1. ホスト特有のスキル（10分）

### オープニングとカウントオフ

**定番パターン（最低3つは身につける）：**

| パターン | 説明 | 使う場面 |
|---------|------|---------|
| ブルース開始 | 曲名→キー→テンポを短く宣言して即カウント | 参加者が多い時の安定運用 |
| スタンダード開始 | フォーム（AABA等）を一言添えてカウント | 初参加者がいる時 |
| ルバート導入 | 2〜4小節の導入後にテンポイン | バラードやミディアム |
| カウントオフのみ | テンポを出して「1, 2, 1-2-3-4」 | アップテンポ |

**達成基準：**

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | 曲名・キー・テンポを明確に伝えてカウントできる |
| ⭐ 実用レベル | 3パターン以上を曲や参加者に応じて使い分けられる |
| 🔥 自動化レベル | その場の編成/空気に合わせて開始方法を即決できる |

---

### 進行キュー（フォーム・ソロ回し・エンディング）

**定番キュー：**

| パターン | 説明 |
|---------|------|
| フォームキュー | Aセクション復帰前やラストA前で短い合図を出す |
| ソロ回しキュー | 「次どうぞ」を明確に伝える（視線・呼称・手） |
| エンディングキュー | タグ、rit.、最終コーラス入りを事前に共有 |

**達成基準：**

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | 2パターンのキューを破綻なく出せる |
| ⭐ 実用レベル | 進行を止めずにフォーム/ソロ/終わりを制御できる |
| 🔥 自動化レベル | キュー出しが自然で、演奏の流れを損なわない |

---

### 参加者サポート・コンピング

**なぜ必要か：**
ホストの役割は「自分が弾くこと」だけではなく、全員を弾きやすくすること。参加者のレベルや楽器編成に応じて、コンピングの密度・音量・音域を調整する必要がある。

**練習の進め方：**
- 初参加者や初心者ソリストには、2拍4拍中心でスペース多めに支える
- 上級者には反応型コンピングで会話を作る
- 迷子になりやすい箇所で、薄くガイドトーンやルート感を補助する

**達成基準：**

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | 相手に合わせて音量・密度を1段階変えられる |
| ⭐ 実用レベル | 参加者の演奏を止めずに1曲通して支えられる |
| 🔥 自動化レベル | その場の相手に合わせて最適な伴奏へ即切替できる |

## 4-2. セッション進行準備・模擬運営（15分）

### 1セットの構成例（5〜6曲、40〜50分）

| 順番 | テンポ/スタイル | 曲の例 | 役割 |
|------|---------------|--------|------|
| 1曲目 | ミディアム・スイング | There Will Never Be Another You | ウォームアップ。参加しやすい曲で開始 |
| 2曲目 | ブルース | Billie's Bounce | 参加者が入りやすい共通言語 |
| 3曲目 | ボサノバ/ラテン | Blue Bossa | 雰囲気を変える |
| 4曲目 | バラード | Body and Soul | 表現の幅を作る |
| 5曲目 | アップテンポ | Take the A Train | 終盤のピーク |
| 6曲目 | ミディアム | Autumn Leaves | 初参加者も戻りやすく締める |

### 模擬運営の方法

1. 曲名・キー・テンポ・フォームを口頭で宣言してから開始する
2. ソロ順とコーラス数の目安を事前共有する
3. 各曲「開始キュー → テーマ → ソロ回し → 終了キュー」を通す
4. 録音し、「説明の明瞭さ」「止まらない進行」「全体時間」を確認する

**達成基準（ホスト開始準備の基準）：**

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | 5曲分の進行を宣言付きで通せる |
| ⭐ 実用レベル | 5〜6曲を止めずに回し、参加者の入退場を整理できる |
| 🔥 実戦運用OK | 現場の人数・編成変化にも対応して自然に進行できる |

## Phase 4 完了＝セッションホスト開始の条件

- [ ] 5〜6曲の進行テンプレート（曲名・キー・テンポ・フォーム）が用意できている
- [ ] 曲の開始/終了/ソロ回しのキューを実地で運用できる
- [ ] 模擬運営を最低3回実施し、録音レビューで改善している
- [ ] 月1回以上、ホストまたはホスト代行として実地で回している
