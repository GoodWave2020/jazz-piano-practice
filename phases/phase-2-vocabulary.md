# Phase 2：ジャズの語彙を増やす（6〜18ヶ月）

## レッスンの活用法

| 時期 | 頻度 | レッスンで見てもらうこと | 独学で進めること |
|------|------|----------------------|---------------|
| 6〜12ヶ月 | 月1回 | A/Bボイシングの響き確認、コンピングのタイム感チェック | ボイシング暗記、スケール練習、曲の譜読み |
| 12〜18ヶ月 | 月2回に増やす | アドリブの方向性指導、リックの使い方、トランスクリプションの分析 | リック暗記、耳コピ、iReal Proでの反復 |

**Phase 2 前半（6〜12ヶ月）で月1回に減らせる理由：**
Phase 1でフォームの基礎が固まっていれば、ボイシングやスケールの練習は独学で十分進められる。コード理論を既に理解しているので、「なぜこの音を押さえるのか」の部分で迷わない。月1回のレッスンでは、独学で練習した内容の「響きのチェック」と「次に何を練習すべきか」の方向性確認に使う。

**Phase 2 後半（12〜18ヶ月）で月2回に戻す理由：**
アドリブの入口は独学で最もつまずきやすいポイント。「スケールは弾けるけど、アドリブにならない」という壁にぶつかるタイミング。ここで先生に「何を考えて弾いているか」を実演してもらい、一緒にアドリブの練習をすることで突破口が見える。これは動画や教本では得られない価値。

**レッスンで特に見てもらうべきポイント：**
- 先生にii-V-Iのアドリブを目の前で弾いてもらい、「何を考えながら弾いているか」を言語化してもらう
- 自分のアドリブを先生に聴いてもらい、「もっとこうした方がいい」の具体的な指摘をもらう
- 先生とのデュオ（先生がベースライン、自分がコンピング＋ソロ）で実践練習

## 2-1. スケール・テクニック（5分）

### ジャズで頻出するスケール

Phase 1のメジャー/マイナーに加えて、以下を順次追加：

| スケール | 用途 | 優先度 |
|---------|------|--------|
| ドリアンスケール | マイナー7thコード上で使用 | 最優先 |
| ミクソリディアンスケール | ドミナント7thコード上で使用 | 最優先 |
| ブルーススケール | ブルース全般 | 高 |
| リディアンスケール | メジャー7thコード上で使用 | 中 |
| オルタードスケール | ドミナント7thの緊張感を出す | Phase 2後半〜 |
| ディミニッシュスケール（コンディミ） | ドミナント7th上の発展的用法 | Phase 2後半〜 |

**練習の進め方：**
- 各スケールをまず3〜4キーで習得 → 全12キーへ
- ii-V-Iの進行上で「Dm7でドリアン → G7でミクソリディアン → CMaj7でリディアン」と繋げて弾く練習が特に重要

**達成基準：**

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | 各スケールの構成音を全12キーで理解している。ゆっくりなら弾ける |
| ⭐ 実用レベル | ドリアン・ミクソリディアンを全12キーで♩=100の8分音符で弾ける |
| 🔥 自動化レベル | ii-V-I進行上でスケールを切り替えながら♩=120で弾ける |

## 2-2. ボイシング＆コンピング（10分）

### A/Bボイシング（ルートレスボイシング）

**A/Bボイシングとは：**
ルート（根音）を省略し、3rd-5th-7th-9th（Aフォーム）または 7th-9th-3rd-5th（Bフォーム）で構成するボイシング。ベースがルートを弾く前提のボイシングなので、トリオ演奏に直結する。

**Cキーのii-V-Iの場合：**

```
【Aフォーム（3-5-7-9）】
Dm7:  F-A-C-E
G7:   B-D-F-A
CMaj7: E-G-B-D

【Bフォーム（7-9-3-5）】
Dm7:  C-E-F-A
G7:   F-A-B-D
CMaj7: B-D-E-G
```

**練習の進め方：**
1. まずAフォームだけで全12キーのii-V-I
2. 次にBフォームだけで全12キーのii-V-I
3. A→B→A→B...と五度圏で交互に弾く
4. 曲のコード進行上で使う

**達成基準：**

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | A/B両フォームを全12キーのii-V-Iで弾ける（テンポ不問） |
| ⭐ 実用レベル | 五度圏で♩=80（1コード2拍）でA/B交互にノンストップ |
| 🔥 自動化レベル | 曲のコード進行上で、適切なフォームを瞬時に選択して弾ける |

---

### コンピングのリズム

**コンピングとは：**
ソリストの演奏に合わせてリズミカルにコードを弾く伴奏スタイル。「いつ弾くか」「いつ弾かないか」が重要。

**基本リズムパターン（優先順）：**

1. **チャールストン・リズム：** 1拍目＋2拍目のウラ（最も基本的なコンピングリズム）
2. **2拍4拍のウラ打ち：** シンプルだがスイング感が出る
3. **シンコペーション：** 小節をまたぐタイ
4. **ランダム：** 上記を組み合わせて即興的に

**練習の進め方：**
1. 1つのコード（例：Dm7）でリズムパターンだけを練習
2. ii-V-Iでコードチェンジしながらリズムパターン
3. iReal Proの伴奏に合わせて曲を通してコンピング
4. 好きなピアニストのコンピングを聴いてリズムを真似る

**達成基準：**

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | チャールストン・リズムで曲を通してコンピングできる（テンポ♩=100） |
| ⭐ 実用レベル | 3種類以上のリズムパターンを混ぜて、♩=140でコンピングできる |
| 🔥 自動化レベル | iReal Proなしで、自分のリズム感だけでスイングしたコンピングができる |

### リック（フレーズ）の習得

**リックとは：**
ii-V-I などの定番コード進行上で使えるお決まりのフレーズ。ジャズの「語彙」にあたる。

**練習の進め方：**
1. 教本やYouTubeからii-V-I リックを1つ選ぶ
2. まず1キー（C）で完全に弾けるようにする
3. 五度圏で全12キーに移調して弾く
4. 曲の中のii-V-Iの場所でそのリックを使ってみる

**リックの選び方：**
- 最初は4〜8音程度の短いものから
- コードトーン（1-3-5-7）中心のシンプルなものから始める
- Red Garland、Wynton Kelly のソロから取るのがおすすめ

**ペース目安：** 2〜3週間に1リック。Phase 2 全体で8〜12リック程度。

**達成基準：**

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | 1キーで正確に弾ける |
| ⭐ 実用レベル | 全12キーで♩=100で弾ける |
| 🔥 自動化レベル | 曲のii-V-Iでそのリックを自然に挿入できる |

---

### トランスクリプション（採譜）

**やること：** 好きなピアニストのソロを耳コピして弾く

**最初に採譜するのにおすすめのソロ：**
- Red Garland — 「Autumn Leaves」のソロ（シンプルでブロックコードが特徴的）
- Wynton Kelly — 「Freddie Freeloader」のソロ（Miles Davisの"Kind of Blue"収録）
- Sonny Clark — 「Cool Struttin'」のテーマ部分

**練習の進め方：**
1. まず繰り返し聴いて、フレーズを歌えるようにする
2. Transcribe!（ソフト）やiPhoneの再生速度変更でテンポを落とす
3. 1フレーズ（2〜4小節）ずつ採譜 → ピアノで弾く
4. なぜそのフレーズが良いのか分析（使っているスケール、アプローチノートなど）

**ペース目安：** Phase 2 全体で1〜2コーラス（32〜64小節）採譜できれば十分

**達成基準：**

| 基準 | 条件 |
|------|------|
| ✅ 最低ライン | 4〜8小節のフレーズを正確に弾ける |
| ⭐ 実用レベル | 1コーラス（32小節）を原曲テンポの70%で弾ける |
| 🔥 自動化レベル | 採譜したフレーズの一部を自分のアドリブに応用できる |

## セッション参加ガイド（Phase 2 中盤〜）

**参加の目安：** A/Bボイシングが⭐実用レベルに達し、5曲以上でコンピングができる状態

**最初のセッションで弾く曲（安全な選択）：**
- Fブルース（Now's the Time, Billie's Bounce）
- Autumn Leaves
- Blue Bossa

**セッションでの心構え：**
- 最初は「コンピングだけで参加」で十分。ソロは求められたら1コーラスだけ
- ミスしても止まらない（ベースで経験済みのはず）
- 他のプレイヤーのソロを聴くことが最大の学び
